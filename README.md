# Multi-Modal-Deep-Learning-Architecture-Fashion-Recommendations-system-Glowup
GlowUp is a hybrid fashion recommendation system that delivers personalized outfit suggestions using user behavior, item similarity, and multimodal features from images and text.
## Overview
GlowUp is an intelligent fashion recommendation system that suggests personalized outfits based on user preferences, behavior, and item similarity.

---

##  Features
- Personalized outfit recommendations  
- Hybrid recommendation system (content-based + collaborative filtering)  
- Color & style matching  
- Outfit generation (complete looks)  
- Explainable recommendations  
- Gender-based filtering  

---

## How It Works
- Uses cosine similarity to find similar fashion items  
- Combines user interactions (view, click, like, purchase)  
- Applies rule-based style and color matching for personalization  

---

##  Tech Stack
- **Backend:** Python, Pandas, Scikit-learn, SQLAlchemy  
- **Frontend:** Angular, TypeScript  
- **Machine Learning:** Cosine Similarity, BERT, Vision Transformer  

---

##  Dataset
- DeepFashion  
- Fashion200k  
- Amazon Fashion Dataset (2020)
